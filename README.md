# Calculadora-de-M-dia-e-Mediana-
É uma calculadora de média(soma de todos os valores de lista de numeros e depois divide pela quantidade de valores na lista) e mediana (em uma lista de ordem de menor para o maior, se for par seleciona o número do meio, e se for impar seleciona os dois valores do meio e divide em 2.

Digite qualquer valor numérico, e precione enter para pular para o proximo número.

Quando desejar digite, "sair" para receber a lista dos número digitados, média e mediana.
